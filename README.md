# customer_behavior_analysis-Python-SQL-PowerBI-
Developed an end-to-end data analytics project analyzing customer behavior using Python, SQL, and Power BI. Performed data cleaning, feature engineering, and SQL-based analysis to uncover trends in customer spending and loyalty. Built interactive dashboards to present insights and support data-driven business decisions.

Customer Behavior Data Analytics Project

**Overview**

This project is an end-to-end Data Analytics solution that analyzes customer shopping behavior to generate meaningful business insights. It uses Python, SQL, and Power BI to perform data cleaning, analysis, and visualization.

The goal of this project is to help businesses understand customer patterns, improve decision-making, and increase revenue through data-driven strategies.

**Objectives**

Analyze customer purchasing behavior
Identify key revenue-driving segments
Evaluate the impact of discounts and promotions
Understand customer loyalty and subscription trends
Build interactive dashboards for business insights

**Tech Stack**

Python → Data cleaning & preprocessing
SQL (PostgreSQL/MySQL) → Data analysis
Power BI → Data visualization & dashboards

**Project Workflow**

1. Data Preparation (Python)
Handled missing values
Standardized column names
Performed feature engineering (age groups, purchase frequency)
Exported cleaned dataset

2. Data Analysis (SQL)
Revenue analysis by gender and category
Discount impact on spending
Top-performing products
Customer segmentation (new, returning, loyal)
Subscription vs non-subscription analysis

3. Data Visualization (Power BI)
Built interactive dashboards
Created KPIs (Total Customers, Revenue, Avg Rating)

Visualized:
Revenue by category
Sales by age group
Subscription distribution
Added filters for dynamic analysis


**Key Insights**

Young adults contribute the highest revenue
Customers using discounts tend to spend more
Low subscription conversion among repeat buyers
Certain product categories drive higher sales


**Business Recommendations**

Target high-value customer segments with focused marketing
Improve subscription conversion strategies
Optimize discount campaigns
Promote high-rated products
